# Лабораторная работа №2. Типы шума на изображении, моделирование аддитивного шума, алгоритмы фильтрации изображений

## Алгоритмы

В процессе выполнения лабораторной работы на языке Python были реализованы следующие алгоритмы:

* Моделирование шума соль-перец
* Фильтр "усреднение" (averaging)
* Медианный фильтр

> Примечание. Имплементация алгоритмов рассчитана, в первую очередь, на обработку изображений с одним каналом (оттенки серого), но при желании ее нетудно обобщить для работы с RGB-изображениями (вопрос только в том, есть ли смысл). Если рассматривать реализацию для RGB-изображений, то, возможно, имеет смысл добавлять не только черные и белые точки, но и другие цвета.

## Наблюдения

* В реализации моделирования шума "соль-перец" при уменьшении нижнего (и увеличении верхнего) порога выборки (30 и 225 соответственно) можно уменьшить степень "зашумленности" изображения.
* Медианный фильтр очень хорошо справляется с удалением шума "соль-перец", а вот усреднение, напротив, только ухудшает изображение, делая его размытым.
